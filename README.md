# Airbnb Clone

This is a prototype of an Airbnb clone developed widt React, Node.js and Material-UI. The project is dockerized for eazy deployment.

## Requirements

- React (v18)
- Node.js (v16)
- Docker

## Configure Docker

 docker build -t airbnb-clone .
 docker run -p 3000:3000 airbnb-clone
